Usage
=====

Quick Start Guide 
-----------------

1.	Setup and load the Scan Protocol Library 
2.	Switch to the Listening Mode:
a.	Load a logging directory (Sec. 4.2)
b.	Connect the instrument (Sec. 4.1)
3.	Run scan protocols (Sec. 5)
4.	Leave the software in Listening Mode or close it.
